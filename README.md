This is a personal portfolio website. It showcases projects, skills, and contact information all in my own design.

## Features

- **Navbar** Easy Navigation Tab
- **Hero Section** Name + Taglines
- **About Section** My Introduction
- **Projects Section** Showcase of my Work 
- **Contact Section** Any relevant Contact Methods
